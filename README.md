The Royal Hotel Management System is a full-stack web application designed to manage hotel operations such as room booking, customer management, staff administration, billing, and reports.
The system is built using Spring Boot (Backend), React.js (Frontend), MongoDB (Database), and deployed on AWS Cloud, ensuring scalability, security, and high availability.

System Architecture

React.js communicates with Spring Boot REST APIs

Spring Boot handles business logic and authentication

MongoDB stores hotel, booking, user, and payment data

AWS EC2 hosts backend services

AWS S3 stores room images and invoices
